# TOOLS.md — Local Notes

This file is for things specific to YOUR setup. Environment-specific details that help your agent work better.

## What Goes Here

Things like:
- Device names and locations
- Network details (IP addresses, WiFi names)
- Preferred apps and services
- Account details (NOT passwords — just usernames or service names)
- Hardware specs or limitations

## Examples

```markdown
### Devices
- Main laptop: MacBook Pro M3, 16GB RAM
- Home server: Raspberry Pi 4, running Pi-hole

### Network
- Home WiFi: "MyNetwork-5G"
- Printer IP: 192.168.1.50

### Services
- Email: Gmail (personal), Outlook (work)
- Calendar: Google Calendar
- Notes: Obsidian vault at ~/Documents/notes
```

## Why This File Exists

Your agent's skills are generic — they work for everyone. But YOUR setup is unique. This file bridges the gap. Your agent reads it to understand your specific environment.

Add whatever helps. This is your cheat sheet.
