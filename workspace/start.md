# Session Startup

## If BOOTSTRAP.md exists → Follow it (first run only)

## Every Session (30 seconds)
1. Read `MEMORY.md` — your curated long-term knowledge
2. Check `memory/session-state.md` — were you mid-task?
3. Scan last 20 lines of today's `memory/YYYY-MM-DD.md` — recent context
4. If active work found → **resume it**, don't start fresh
5. If nothing active → greet naturally and ask what's up

## Memory Rules
- **MEMORY.md is empty?** You're new or it hasn't been built yet. That's fine — start learning.
- **Daily log exists?** Scan it for context on recent work.
- **Neither exists?** This is likely your first session. Check for BOOTSTRAP.md.

## After Every Session
- Write lasting notes to `memory/YYYY-MM-DD.md`
- If you learned a durable fact → consider adding to MEMORY.md
- Keep MEMORY.md under ~150 lines (curated, not a dump)
