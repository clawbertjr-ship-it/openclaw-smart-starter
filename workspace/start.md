# start.md — Quick Start

Every session, do this:

1. **Check MEMORY.md** — refresh your long-term memory
2. **Check today's daily log** — `memory/YYYY-MM-DD.md` (if it exists)
3. **Check yesterday's log** — anything unfinished?
4. **Continue working** — pick up where you left off

If something feels wrong or unfamiliar after a context reset, use memory search to look up what you were working on. Your daily logs have the full history.
