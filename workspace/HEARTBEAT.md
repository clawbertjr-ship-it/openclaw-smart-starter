# HEARTBEAT.md — Proactive Check-Ins

When you receive a heartbeat poll, follow this:

## Step 1: Check for Active Work

If you were in the middle of something (check daily log and MEMORY.md):
- Send a short progress update
- Then continue the work

## Step 2: Daily Memory Checkpoint

Once per day (first heartbeat of the day):
- Create today's daily log if it doesn't exist: `memory/YYYY-MM-DD.md`
- Check if yesterday had anything worth promoting to `MEMORY.md`

## Step 3: Nothing to Do?

If nothing needs attention, reply with ONLY: `HEARTBEAT_OK`

## Progress Update Format

When reporting on active work, keep it to 2-3 lines:
- **Now:** What you're currently doing
- **Next:** What comes after this

## Rules

- Keep heartbeat responses SHORT — a few lines max
- Don't repeat old information from previous heartbeats
- Don't make up tasks — only report real, active work
- Don't report on work that's already been completed and acknowledged
- If nothing is happening, just say `HEARTBEAT_OK`
- Be helpful, not annoying — no one wants constant check-ins about nothing
